# Eidra Social Content Pack

Use these drafts as starting points for X, LinkedIn, GitHub posts, Discord communities, or launch threads.

## Posting Rules

- lead with the problem and the concrete category
- keep the first post simple
- link to one primary destination
- pair copy with a screenshot or demo GIF when possible
- do not over-claim

## English Short Post

```text
AI development needs a localhost trust layer.

I built Eidra, an open-source local proxy and MCP firewall that helps you see, control, and explain what AI tools are sending out of your machine.

If you use Cursor, Claude Code, SDK workflows, or MCP tools, I would love feedback.
```

## English Longer Post

```text
Modern AI development is not just prompts anymore.

It is editors, coding agents, SDKs, MCP tools, and CI workflows. That means more context leaving the machine and more tool access being granted, often with very little visibility.

I built Eidra to be a local-first trust layer for that boundary.

It can scan outgoing requests, apply local policy, mask or block sensitive content, route some sensitive requests locally, and give you a live dashboard plus machine-readable output for automation.

If you are working with AI coding tools or MCP workflows, I would love honest feedback.
```
