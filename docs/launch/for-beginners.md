# Eidra — AIを安全に使うための「見張り番」

## AIツール、便利だけど怖くない？

CursorやCopilot、Claude Codeを使っていると、こんなことが起きています：

- あなたのパスワードが書いてあるファイル（`.env`）を、AIが勝手に読んでクラウドに送っている
- 顧客のメールアドレスや電話番号が、AIの学習データに入ってしまう可能性がある
- 最近のAIは「コマンドを実行する」こともできるので、間違えてファイルを全部消すこともありうる

でも、何が送られているか見えないから、気づけない。

---

## Eidra（エイドラ）って何？

**あなたのパソコンの中で動く、AIの見張り番です。**

AIツールとインターネットの間に立って、出ていくデータを全部チェックします。

やることは3つだけ：

### 1. 見せる
AIが裏で何を送っているか、リアルタイムで画面に表示します。

```
23:41:05  ✓ ALLOW   api.openai.com       clean
23:41:08  ◐ MASK    api.anthropic.com     2 findings
                     ├─ api_key
                     └─ email_address
23:41:12  ✗ BLOCK   api.openai.com       1 finding
                     └─ private_key
```

今まで見えなかったものが、全部見える。それだけで安心感が全然違います。

### 2. 隠す
パスワードやAPIキーを見つけたら、AIに送る前に自動で隠します。

```
あなたが書いたコード:
  api_key = "sk-abc123secret456"

AIに届くもの:
  api_key = "[REDACTED:api_key:a3f2c1]"
```

AIには隠した状態で届くので、普通に回答してくれます。あなたは何も気にせずAIを使えます。

### 3. 止める
AIが危険なことをしようとしたら、実行される前にブロックします。

```
AIが実行しようとしたこと:
  → DROP TABLE users（データベースを全消去）
  → rm -rf /（ファイルを全部削除）
  → cat ~/.ssh/id_rsa（秘密鍵を読み取り）

Eidraの対応:
  → 「ブロックしました」（実行されません）
```

---

## 使い方

ターミナルで3行打つだけです。

```
eidra init
eidra start
```

これだけ。あとは裏で勝手に守ってくれます。

ダッシュボード（監視画面）を見たいときは：

```
eidra dashboard
```

## よくある質問

**Q: AIの動作が遅くならない？**
ほぼ変わりません。チェックにかかる時間は0.005秒以下です。

**Q: 設定は必要？**
不要です。最初から47種類のチェックルールが入っています。

**Q: お金はかかる？**
無料です。オープンソース（MIT License）なので、コードも全部公開されています。

**Q: データが外部に送られない？**
はい。Eidra自体は一切外部と通信しません。すべてあなたのパソコンの中だけで動きます。

**Q: どのAIツールで使える？**
Cursor、Claude Code、GitHub Copilot、ChatGPT API、その他すべてのAI APIツールで動きます。

あなた  ＝ 社長

          「この書類は社外秘なので、名前だけ消してから渡しますね」
